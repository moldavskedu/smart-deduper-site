Smart Deduper Google Sheets add-on

Fast, safe duplicate cleanup. Keeps one row per group in black, marks the rest red. Works across selected sheets, respects first-name differences, blocks 1-digit phone neighbors, and honors Customer/Serial IDs.

Install from Google Workspace Marketplace

  • Strong IDs (Email / Phone / Customer ID / Serial)
  • “Always keep one” enforcement + clear coloring
  • Results & Log pop-outs, Reset colors, Advanced options
  • Zero deletion unless you confirm

Privacy-friendly
